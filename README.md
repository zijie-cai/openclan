OpenCLAN is a modern, web-based platform for computational linguistic analysis using the CHAT (.cha) format.

Generate structured transcripts.  
Build %mor, %gra, and %err tiers.  
Compute MLU, TTR, FREQ, and CoreLex.  

Fully editable. Fully compatible. Research-ready.

If you want a streamlined, AI-assisted workflow for CLAN-style analysis — this is it.

## Why OpenCLAN?

CLAN is powerful — but its workflow is fragmented and outdated.

OpenCLAN modernizes the process:

• Native `.cha` transcript generation  
• Automatic linguistic tier construction  
• Live metric computation  
• Speaker diarization from audio  
• Editable transcript interface  
• Strict CHAT compliance  

Everything remains researcher-controlled.

No proprietary lock-in.  
No black-box annotation.

---

## How It Works

1. Upload audio  
2. Automatic transcription + speaker tagging  
3. Generate %mor / %gra / %err tiers  
4. Edit transcripts interactively  
5. Analyze sessions instantly  
6. Export `.cha`, `.csv`, or `.json`

From raw audio to structured linguistic analysis — in one workflow.
